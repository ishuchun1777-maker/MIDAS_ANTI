# Agent Rules

## Rule 1: Aesthetic Excellence
- Every UI element must be visually stunning.
- Use smooth transitions and micro-animations.
- Avoid browser defaults.

## Rule 2: Non-Destructive Changes
- Never delete core logic without a backup or explicit instruction.
- Prefer refactoring over rewriting from scratch.

## Rule 3: Comprehensive Documentation
- All new features must be reflected in the `/docs` folder.
- Maintain the `/agents` files to reflect new standards.
