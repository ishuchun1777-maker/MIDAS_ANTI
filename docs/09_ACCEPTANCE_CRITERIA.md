# Acceptance Criteria - MIDAS

## MVP Requirements
1. **Authentication**: All users must be verified via Telegram InitData.
2. **Navigation**: Deep linking via `startapp` must work flawlessly.
3. **UI**: Glassmorphism and Framer Motion must be present in core screens.
4. **Deals**: Multi-step flow must support status updates.
5. **Chat**: Messages must deliver in real-time (polling or sockets).
