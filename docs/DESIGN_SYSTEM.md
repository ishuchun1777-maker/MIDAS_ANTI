# MIDAS Design System

MIDAS is a premium advertising marketplace platform.

Design goal:

Create a modern, minimal and premium SaaS interface.

The UI should feel similar to:

- Stripe
- Linear
- Vercel
- Notion (dark mode)

The platform must look professional and trustworthy for business users.

# Typography

Use modern sans-serif fonts.

Recommended:

Headings:
Inter / Satoshi / Geist

Body:
Inter / System font

Avoid decorative fonts.

---

# Spacing System

Use consistent spacing scale:

4px
8px
12px
16px
24px
32px
48px

---

# Border Radius

Small: 6px  
Medium: 10px  
Large: 16px

---

# Shadows

Cards should have soft shadows.

Avoid heavy drop shadows.

---

# Icon Style

Use clean outline icons.

Recommended libraries:

- Lucide
- Heroicons
- Tabler icons

---

# Layout

The platform must use a dashboard layout:

Sidebar navigation  
Top header  
Main content area

---

# Interaction Style

Animations should be subtle.

Hover states must exist.

Transitions should be smooth but fast.

---

# Design Philosophy

MIDAS should look like a high-end tech platform for advertising professionals.
