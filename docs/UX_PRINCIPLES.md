# MIDAS UX Principles

MIDAS is used by entrepreneurs, marketers and media owners.

Users must be able to understand the platform within seconds.

---

# Principle 1: Simplicity

Avoid complex workflows.

Each action should be obvious.

---

# Principle 2: Clarity

Always show:

- status
- progress
- next step

Users should never feel lost.

---

# Principle 3: Fast Actions

Important actions must be accessible quickly.

Example:

Send offer
Accept offer
Upload proof

These actions should be visible without deep navigation.

---

# Principle 4: Transparency

Users must clearly see:

campaign status  
deal status  
offer status  

Example statuses:

pending  
accepted  
rejected  
completed

---

# Principle 5: Trust

The platform must build trust.

Use:

verification badges  
ratings  
reviews  

---

# Principle 6: Business Focus

MIDAS is not a social network.

It is a business tool.

Avoid unnecessary features.

Focus on deals and results.

---

# Principle 7: Mobile Friendly

Mini App must be optimized for mobile.

Web platform must be optimized for desktop dashboards.

---

# Principle 8: Feedback

Every action must have feedback.

Example:

success messages  
error messages  
loading states
