# User Flow Blueprint - MIDAS

## Core Flow: Purchasing an Item
1. User interacts with Telegram Bot button with `startapp=listing_123`.
2. Mini App opens and navigates directly to Listing `123`.
3. User clicks "Start Deal".
4. Multi-step deal UI opens.
5. Users chat in real-time within the deal UI.
6. Seller marks as "Completed".
7. Buyer leaves a rating and review.
