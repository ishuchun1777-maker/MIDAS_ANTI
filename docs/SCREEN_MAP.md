# MIDAS Screen Map

This document defines all screens in the MIDAS platform.

The platform has three main interfaces:

1. Web Dashboard (desktop SaaS interface)
2. Telegram Mini App (mobile interface)
3. Telegram Bot (entry point + notifications)
