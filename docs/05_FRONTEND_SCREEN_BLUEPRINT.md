# Frontend Screen Blueprint - MIDAS

## Key Screens
1. **Home/Marketplace**: Masonry or grid layout with dynamic filtering.
2. **Listing Details**: High-resolution image view, description, and "Start Deal" button.
3. **Deal Dashboard**: Multi-step flow indicator and embedded chat.
4. **User Profile**: Display stars, reviews, and active listings.
5. **Loading/Splash**: Premium glassmorphism animations.
