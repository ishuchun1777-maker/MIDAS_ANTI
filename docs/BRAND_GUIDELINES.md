# MIDAS Brand Guidelines

MIDAS is a premium advertising marketplace platform.

Design principles:

- modern
- minimal
- premium
- tech startup style
- dark interface

---

Primary color

#00C27C
(neon emerald)

Usage:
primary buttons
active elements
links

---

Accent color

#FFB800
(bright gold)

Usage:
highlights
premium elements
badges

---

Background

#080C12
(dark navy)

---

Card background

#0F1923

---

Borders

#1A2535

---

Text color

#F0F4F8

---

Muted text

#4A5568

---

UI style

Use modern startup style similar to:

- Stripe
- Linear
- Vercel
- Notion dark theme

---

Typography

Headings: bold modern sans-serif  
Body text: clean readable sans-serif

Avoid decorative fonts.

---

UI Components

Rounded corners: medium  
Cards: soft shadows  
Buttons: solid primary color

---

Design goal

MIDAS should feel like:

"Silicon Valley level SaaS platform for advertising industry."
